### Опыт и проекты
* **Backend-разработчик (Расчет энергопотребления):** Внедрил чистую архитектуру на Golang, разделив систему на независимые сервисы. Настроил сетевое взаимодействие через gRPC и работу с базой данных через GORM.
* **Хакатон Comet Tracker:** Разработал Go-сервер с REST API. Интегрировал PostgreSQL, Redis для сессий и MinIO для хранения изображений.
* **Telegram-бот для игры в слова:** Создал асинхронный сервис на Python (Aiohttp). Реализовал модель данных через SQLAlchemy и управление игровыми сессиями с рейтинговой системой.
* **Desktop-приложение магазина игр:** Спроектировал многослойную архитектуру на Java. Реализовал интерфейс на JavaFX и многопоточную загрузку данных.
